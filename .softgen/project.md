# Developer Portfolio — Eswar Chinthakayala

## Vision
A modern, professional portfolio site showcasing full-stack development expertise. Clean, tech-forward design that demonstrates technical authority without pretense. Target audience: recruiters, potential clients, collaborators.

## Design
Color tokens (HSL):
- `--primary: 187 95% 43%` (bright cyan)
- `--background: 0 0% 98%` (subtle off-white)
- `--foreground: 222 47% 11%` (deep slate)
- `--accent: 187 95% 43%` (cyan, same as primary)
- `--muted: 220 13% 91%` (light slate)

Typography:
- Headings: Plus Jakarta Sans (600, 700)
- Body: Work Sans (400, 500, 600)

Style: Modern tech aesthetic — generous whitespace, card-based layouts, subtle shadows for depth. Mobile-first responsive grid. High contrast for readability.

## Features
- Hero section with name, title, brief intro
- Skills showcase organized by category (Frontend, Backend, Database, Design)
- Contact/social links
- Responsive design for all screen sizes
